# Linux Basic Commands

This file contains essential Linux commands used in cybersecurity and daily terminal operations.
## Navigation Commands

### pwd
Shows current directory
Example:
pwd

### ls
Lists files and directories
Example:
ls
ls -la

### cd
Change directory
Example:
cd /home
cd ..
## File Operations

### touch
Create new file
Example:
touch file.txt

### mkdir
Create directory
Example:
mkdir folder

### rm
Delete files
Example:
rm file.txt
rm -r folder
## Viewing Files

### cat
Display file content
Example:
cat file.txt

### less
View large files
Example:
less file.txt
## Permissions

### chmod
Change file permissions
Example:
chmod 755 file.sh

### chown
Change ownership
Example:
chown user file.txt
## Networking Commands

### ping
Check connectivity
Example:
ping google.com

### ifconfig
Show network interfaces
Example:
ifconfig

### netstat
Show network connections
Example:
netstat -tuln
## Pro Tips

- Use `clear` to clean terminal
- Use arrow keys to access previous commands
- Use Tab for auto-completion
