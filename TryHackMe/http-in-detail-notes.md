# HTTP in Detail - TryHackMe

## What I Learned
- HTTP request methods: GET, POST, PUT, DELETE
- GET - retrieve data from server
- POST - send data to server (login, forms)
- PUT - update existing data
- DELETE - delete data

## Key Headers
- Content-Type - tells browser what type of data is returned
- Set-Cookie - saves cookies to your computer

## HTTP Response Codes
- 200 - OK (success)
- 404 - Not Found
- 403 - Forbidden

## What I Practiced
- Made GET, POST, PUT, DELETE requests
- Logged in using POST with username/password
- Used body and URI parameters
